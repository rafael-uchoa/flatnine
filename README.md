## Installation

### Back-end

- `cd backend`
- `npm i || yarn`
- `npm run dev || yarn dev`

### Front-end

- `cd frontend`
- `npm i || yarn`
- `npm run start || yarn start`

### Mobile

- `cd frontend`
- `npm i || yarn`
- `npm run start || yarn start`
